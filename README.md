# Blockchain-Based Insurance Platform

## Overview
This project is a **Blockchain-Based Insurance Platform** that automates the insurance claims process using **smart contracts**, ensuring **transparency, security, and faster payouts**.

## Features
- **Automated Claims Processing:** Smart contracts handle claims seamlessly.
- **Enhanced Security:** Blockchain ensures data integrity and prevents fraud.
- **Faster Payouts:** Real-time verification leads to quicker claim settlements.
- **Transparency:** Immutable records provide full auditability.


## Technologies Used
- **Blockchain:** Ethereum, Hyperledger Fabric
- **Smart Contracts:** Solidity
- **Backend:** Node.js
- **Frontend:** React.js
